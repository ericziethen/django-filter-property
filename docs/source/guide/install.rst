============
Installation
============

## Requirements

* Python 3.8
* Django-filter 2.20

## Installation

Install using pip:

.. code-block:: python

    pip install django-property-filter

Then add 'django_property_filter' to your INSTALLED_APPS.

.. code-block:: python

    INSTALLED_APPS = [
        ...
        'django_property_filter',
    ]
